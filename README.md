# AndBible: Bible Study

A powerful, yet easy to use, offline Bible study application.

> [!note]
> This repository is the iOS-targeted AndBible codebase. It carries the same AndBible project forward on iOS while staying aligned with the main Android repository: https://github.com/andbible/and-bible

## 🏗️ Developers

Do you know iOS development, Swift, C, Javascript, or Vue.js? These are the main technologies used in this repository. Pull requests are welcome.

For the iOS codebase, start with [QUICKSTART.md](QUICKSTART.md) and [docs/README.md](docs/README.md). For the main Android repository and broader project context, see [andbible/and-bible](https://github.com/andbible/and-bible).

## 💬 Translators

Everyone is welcome to help translate AndBible to different languages via easy to use web-based Transifex tool. Please see our [Wiki page](https://github.com/AndBible/and-bible/wiki/Translating-User-Interface) to get started.
